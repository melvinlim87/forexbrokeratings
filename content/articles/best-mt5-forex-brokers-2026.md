## MT5 Migration Tips

If you're moving from MT4 to MT5, keep these points in mind:

1. **EAs need porting** — MQL4 code won't run natively on MT5. You'll need to rewrite EAs in MQL5 or find MQL5 equivalents on the marketplace.
2. **Indicator compatibility** — Custom MT4 indicators need to be converted to MQL5. Many popular indicators already have MQL5 versions available.
3. **History data differs** — MT5 uses a different historical data format. Your MT4 backtesting data won't transfer directly.
4. **Account types may differ** — Some brokers offer different leverage, spreads, or instrument availability on MT5 vs. MT4.
5. **Use the strategy tester** — MT5's multi-threaded, multi-currency strategy tester is significantly more powerful than MT4's. Take advantage of it.
